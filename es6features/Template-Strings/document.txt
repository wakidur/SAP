/*********** ** Template Strings/ Template literals ******************** */
1. Template Strings/ Template literals : Template literals are string literals allowing embedded expressions. you can use multi-line strings and string interpolation features with them. They ware called "template strings" in prior editions of the ES2015 specification.

2. Syntax 
    a => `string text`
    b => `string text line 1
          string text line2`
    c => `string text ${expression} string text`
    d => `string text ${expression} string text`

3. Description : 
    3.1 Multi-line strings
    3.2 Expression interpolation
    3.3 Expression interpolation
    3.4 Tagged templates
