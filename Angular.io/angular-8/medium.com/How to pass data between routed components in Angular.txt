There are several ways how Angular components can pass data around:

    2. By injecting parent component through constructor or child components through @ViewChild, @ViewChildren, @ContentChild, @ContentChildren and directly calling componentâ€™s API
    3. Using services (this covers state management libraries like ngrx)
